Substring

Defined in functions_string.yaml

Extract a substring of a specified `length` starting from position `start`. A `start` value of 1 refers to the first characters of the string.

Kernels ¶

  • substring(string, i32, i32) -> string : [] (not supported by dialect)
  • substring(fixedchar, i32, i32) -> string : [] (not supported by dialect)
